# SisRestaurante-TDD

Proyecto desarrollado en Python aplicando pruebas unitarias y pruebas de integración mediante la metodología TDD (Test Driven Development), conectado a una base de datos MySQL.

## Descripción

El sistema simula la gestión básica de un restaurante utilizando arquitectura por capas (DAO, entidades, servicios y pruebas). Durante el desarrollo se implementaron y corrigieron funcionalidades relacionadas con clientes, platos, órdenes y fidelización.

## Tecnologías utilizadas

- Python 3.13
- Pytest
- Pytest-Mock
- MySQL / phpMyAdmin
- MySQL Connector
- Git y GitHub
